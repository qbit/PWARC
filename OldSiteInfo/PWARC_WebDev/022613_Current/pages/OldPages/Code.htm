MORSE CODE

Morse Code has been used for well over 100 years as an efficient way to transmit information from one location to another, and until a few years ago every ham had to learn and demonstrate his skill in sending and receiving Morse Code at a prescribed speed depending on the license class being sought. While knowing the code is no longer required for amateur radio licenses, it has gained more popularity than ever in the ham community.

Learning the code is not difficult, but one must have the discipline to practice it daily to gain speed. There are many ways to teach the code, but the information below is MY way of doing it. If you find another way that is easier for you, feel free to use it.

Code is made up of only two elements, the "dit" (.) and the "dah" (-). The dit is the shortest element you can send, and the dah should be three times as long as a dit. By leaving the "t" off the "dit", unless it is the last "dit" in a letter, your speed will increase faster. For example, the letter "A" is "di-dah", not "dit dah." The letter "B" is "dah-di-di-dit." Practice each session below until you know it thoroughly. Try making words using each session you have learned before moving on to the next session. Then start combining letter from the sessions to make words and then sentences.

The best way to learn the code is to with the help of a buddy who knows or is learning the code as well. If you have to do it on your own though, it can be done. A key and code practice oscillator are optional to learn the code, but they can be an invaluable aid. Code tapes, CDs or on-line code practice sessions are great too, but first you have to get your speed up to the speed at which they begin. Tap the code out with the eraser end of a pencil. Translate signs along the highway into code in your head, and do the same with newspaper headlines. The hardest part for me were letter/number combinations one after the other, and groupings of numbers. If you have the same problem, make up lists of call signs, real or imaginary, and practice them extensively. For number groupings, dig out the telephone directory and practice from it.

Some find the code almost overwhelming at first, but make a game of it and practice, practice, practice. You'll soon find you have learned the code. Then you just need to practice, practice, practice to improve your speed.

HAVE FUN!

Session 1

Let's begin by learning the 4 letters made only of dits (e, i, s, h, 5):

    . = E

    .. = I

    ... = S

    .... = H

Session 2

Next, let's learn the 3 letters made only of the "dah" element (t, m o, 0):

    - = T

    -- = M

    --- = O

 

Session 3

Thngs have been easy so far, right? Now let's start combining the dits and dahs.

    .- = A

    .-- = W

    .--- = J

Session 4

    ..- = U

    ...- = V

Session 5

    -. = N

    -.. = D

    -... = B

Session 6

    --. = G

    --.. = Z

Session 7

Opposites

    .-. = R

    -.- = K

Session 8

More opposites

    .--. = P

    -..- = X

Session 9

C and Q go together, as CQ is what you call when you want to make a contact! L is the reflection of Q.

    -.-. = C

    --.- = Q

    -.-- = L

 

Session 10

F and Y are reflections of each other

    ..-. = F

    -.-- = Y

 

Session 11

The first 5 letters begin with "dit"

    .---- = 1

    ..--- = 2

    ...-- = 3

    ....- = 4

    ..... = 5

Session 12

The last 5 letters begin with "dah"

    -.... = 6

    --... = 7

    ---.. = 8

    ----. = 9

    ----- = 0

Session 13

Punctuation Marks

    .-.-.- = .

    --..-- = ,

    ..--.. = ?

    -..-. = /

    -...- = Dash/Pause

Session 14

Special Characters

    .-.-. = AR (end of message)

    .-... = AS (please wait)

    -...-.- = BK (break)

    ...-.- = SK (end of QSO)

    ........ = Error
