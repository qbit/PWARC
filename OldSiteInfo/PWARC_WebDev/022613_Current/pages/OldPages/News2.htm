Program:

  1. KB0JIT presented a very informative program on how to set up an emergency backup battery system. Paul also handed out schematics of the circuits he used to build both a low power backup system (15 amps) and a high power system (25 amps or more). The systems can be set up to charge batteries when commercial power is available, and these batteries in turn run 12-volt devices. They can be built to automatically switch radios and other 12-volt accessories to battery power when the commercial power goes off.
